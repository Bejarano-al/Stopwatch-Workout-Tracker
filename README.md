# Stopwatch-Workout-Tracker
A Python-based program used track the time one works out (exercise time - rest time) and saves the workout data automatically

# How It Works
-Click Start Workout  
-Track exercise and rest times
-Add optional notes
-Save the workout
-Generate monthly reports
-Review reports in folder  

# Created as a learning project to understand
-Tkinter GUI  
-File handling (JSON)  
-Date & time tracking  
-Simple reporting system  
-Threads & timers
